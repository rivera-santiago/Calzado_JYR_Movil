Testing (opcional)

Instalación (sólo si quieres ejecutar tests localmente):

```powershell
pnpm add -D vitest @testing-library/react-native @testing-library/jest-native react-test-renderer
```

Ejecutar tests:

```powershell
pnpm vitest
```

Nota: El repositorio no tiene estos devDependencies instalados por defecto; añádelos sólo si quieres ejecutar las pruebas.
